# Café Soumya ☕

An interactive, pixel-art coffee-shop-themed portfolio for Soumya Sri Perepu — PhD candidate in Computer Science at Pace University, researching health informatics and cancer-focused ML.

Skills, projects, experience, and education are served up café-style: a menu board, pinned order tickets, a staff schedule, a diploma wall, and a tip jar for contact links. A quest-log HUD tracks which stations you've visited.

Single self-contained HTML file — open `index.html` in any browser, no build step required.
